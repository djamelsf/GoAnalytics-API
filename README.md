# GoAnalytics-API
<p> 
To launch the API you must first install the following libraries:: </p>

```
pip install Flask
```
```
pip install Flask-RESTful
```
```
 pip install requests
```
```
 pip install spacy
```
```
python -m spacy download en_core_web_lg
```

Then run the API : 

```
 flask run
```
